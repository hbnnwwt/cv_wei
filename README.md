# 计算机视觉课程 - 自动阅卷系统

## 项目简介

本课程是一个11周的计算机视觉通选课，以"自动阅卷系统"为项目导向，采用OBE（Outcome-Based Education）理念设计。学生将通过构建一个完整的AI助教系统，学习从图像处理到文字识别的全套计算机视觉技术。

## 课程特色

- **项目驱动**：以自动阅卷系统为核心项目，贯穿整个课程
- **故事化教学**：每周解决一个实际问题，逐步构建完整系统
- **AI辅助编程**：降低编程门槛，专注CV技术学习
- **渐进式能力培养**：从基础到项目，难度逐步提升

## 课程大纲

| 周次 | 主题 | 核心内容 |
|------|------|----------|
| 1 | 计算机视觉导论与图像基础 | OpenCV基础、像素操作、图像滤镜 |
| 2 | AI辅助编程工具实战 | Prompt工程、代码生成、调试技巧 |
| 3 | 图像预处理与增强 | 去噪、二值化、透视矫正 |
| 4 | 试卷版面分析 | 边缘检测、轮廓分析、区域定位 |
| 5 | 选择题识别 | 填涂检测、OMR识别 |
| 6 | 判断题识别 | 符号匹配、形状识别 |
| 7 | OCR基础与文字识别 | PaddleOCR印刷文字识别 |
| 8 | 手写简答题识别 | TrOCR/PaddleOCR手写识别 |
| 9 | 系统架构与分组开发 | 模块化设计、团队协作 |
| 10 | 核心开发与调试 | 系统集成、测试优化 |
| 11 | 成果展示与总结 | 项目展示、课程回顾 |

## 项目功能

自动阅卷系统支持三种题型的自动识别：

1. **选择题**：通过OMR（光学标记识别）技术识别填涂选项
2. **判断题**：通过符号匹配识别√和×
3. **简答题**：通过OCR技术识别手写文字内容

## 技术栈

- **图像处理**：OpenCV
- **文字识别**：PaddleOCR、TrOCR
- **编程语言**：Python
- **开发工具**：AI辅助编程（ChatGPT/Claude等）
